# Smart Lock Admin Dashboard

This is the Capstone Project of Bahcesehir University.
In this application We applied;

- React.js
- Material UI
- Redux

------
### Installation

  
```

$ git clone https://github.com/akasaa101/SmartLockAdminDashboard.git

$ cd SmartLockAdminDashboard

$ yarn install

$ yarn start

```
